# Guías Didácticas Interactivas

## Propósito

El conocimiento es un derecho fundamental, no un privilegio. Cuando ese conocimiento se comparte de forma libre y accesible, se convierte en una herramienta de transformación social. Estas guías nacen de esa convicción: cada una es una **cápsula informativa portable** — un archivo HTML que carga consigo todo lo necesario para funcionar sin servidores, sin instalaciones, sin dependencias.

No son simplemente textos que se leen. Son espacios que se exploran. El diseño visual está pensado para invitar a la interacción: manipular código, observar qué cambia, romper cosas y aprender de los resultados. Porque aprender haciendo no es solo un método más eficaces — es también más memorable, más divertido, y más libre.

## Cómo se generan

El proceso combina inteligencia artificial con revisión humana para crear contenido riguroso y visualmente cuidado:

1. **Esquema inicial con DeepSeek** — se solicita un outline estructurado del tema, organizando los conceptos clave y su secuenciación pedagógica
2. **Desarrollo con Claude** — el outline se convierte en el HTML completo con CSS y JS embebidos, aplicando el diseño visual y la interactividad
3. **Revisión manual** — se verifican errores técnicos (HTML, CSS, JS) y de concepto (contenido, coherencia pedagógica, claridad expositiva)

Este flujo permite producir guías de forma iterativa sin sacrificar la calidad del contenido ni la consistencia visual.

## Uso

**Libre y gratuito.** Sin copyrights restrictivos, muros de pago o necesidad de crear una cuenta para acceder. El conocimiento debe ser de quien lo necesite.

De este modo, se te invita a:
- **Compartir** las guías con quien pueda necesitarlas
- **Crear nuevas** siguiendo este formato — el HTML autocontenido facilita que cualquiera pueda partir de una guía existente y adaptarla a otra materia
- **Adaptarlas** a diferentes niveles, contextos o idiomas

Porque la formación no debería tener barreras de entrada. Y porque el progreso social real acontece cuando quienes saben comparten lo que saben con quienes quieren aprender.
