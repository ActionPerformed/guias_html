Estructura de un Script

Todo script .bat bien escrito sigue esta estructura mínima:

estructura_basica.bat 
@echo off                 :: Oculta los comandos al ejecutarse
:: ────────────────────────────────
:: Aquí va el código del script
:: ────────────────────────────────
pause                     :: Espera pulsación antes de cerrar
exit /b                   :: Termina el script limpiamente
VARIABLES — sintaxis
:: Declarar y leer
set nombre=Juan
echo Hola %nombre%

:: Pedir al usuario
set /p respuesta="¿Continuar? "

:: Aritmética
set /a total=%1 + %2
CONDICIONALES + BUCLES
:: if básico
if "%var%"=="S" echo Sí

:: if/else
if exist "f.txt" (
  echo Existe
) else (
  echo No existe
)

:: "Bucle" con goto
:inicio
set /p v="Valor: "
if "%v%"=="" goto :inicio

Apagado Voluntario

shutdown · if /i · exit /b
RESUELTA
Pregunta al usuario si desea apagar el equipo (S/N). Si responde S → apaga a los 10 segundos. Si responde N → cierra sin hacer nada. Si otro valor → avisa del error.
apagar.bat 
@echo off
set /p respuesta="¿Desea apagar el equipo? (S/N): "
if /i "%respuesta%"=="S" (
    echo Apagando el equipo en 10 segundos...
    shutdown /s /t 10
) else if /i "%respuesta%"=="N" (
    echo No se apagará el equipo.
    exit /b
) else (
    echo Opción no válida. Ejecute de nuevo el programa.
)

Presentación Personal con Validación

set /p · goto · if · robustez
RESUELTA
Pide nombre, apellidos y edad. Muestra: "Hola [nombre] [apellidos], tienes [edad] años." Si el usuario no introduce algún campo, el script debe repetir la pregunta informando del error.
presentacion.bat 
@echo off

:pedir_nombre
set /p nombre="Introduce tu nombre: "
if "%nombre%"=="" (
    echo No introdujiste ningun nombre. Intentalo de nuevo.
    goto pedir_nombre
)

:pedir_apellidos
set /p apellidos="Introduce tus apellidos: "
if "%apellidos%"=="" (
    echo No introdujiste ningun apellido. Intentalo de nuevo.
    goto pedir_apellidos
)

:pedir_edad
set /p edad="Introduce tu edad: "
if "%edad%"=="" (
    echo No introdujiste ninguna edad. Intentalo de nuevo.
    goto pedir_edad
)

echo.
echo Hola %nombre% %apellidos%, tienes %edad% años.
pause
// El patrón goto funciona como un bucle: si la condición no se cumple, vuelve a la etiqueta (:pedir_nombre). No existe while nativo en Batch — este es el idiom equivalente.

Borrado Seguro de Fichero

%1 · if exist · del · validación
RESUELTA
Script borrar.bat que recibe un archivo como parámetro (%1). Debe comprobar: 1) que se pasó parámetro, 2) que el archivo existe. Si existe → lo borra. Si no → informa.
borrar.bat 
@echo off

if "%1"=="" (
    echo ERROR: No se ha introducido ningun parametro.
    echo Uso correcto: borrar.bat "nombre_del_archivo"
    exit /b
)

if exist "%1" (
    echo El archivo "%1" existe. Procediendo a borrarlo...
    del "%1"
    echo Archivo borrado correctamente.
) else (
    echo ERROR: El archivo "%1" no existe en el sistema.
)
pause

Calculadora de Suma por Parámetros

%1 · %2 · set /a · validación doble
RESUELTA
Script suma.bat que recibe 2 números como parámetros y devuelve su suma. Ejemplo: suma.bat 4 5 → "El resultado de sumar 4 y 5 es: 9"
suma.bat 
@echo off

if "%1"=="" (
    echo ERROR: Debes introducir dos numeros.
    echo Uso: suma.bat numero1 numero2
    exit /b
)
if "%2"=="" (
    echo ERROR: Faltan parametros. Debes introducir dos numeros.
    exit /b
)

set /a resultado=%1 + %2
echo El resultado de sumar %1 y %2 es: %resultado%
pause

P1

Comprobador de Paridad

%1 · set /a · %% · equ
PROPUESTA
Crea paridad.bat que reciba un número como parámetro y muestre si es par o impar. Comprueba que se introdujo parámetro.
Operador módulo en Batch: set /a resto=%1 %% 2 — si resto equ 0 → PAR
paridad.bat — SOLUCIÓN 
@echo off
if "%1"=="" (
    echo ERROR: Debes introducir un numero.
    echo Uso: paridad.bat numero
    exit /b
)

set /a resto=%1 %% 2
if %resto% equ 0 (
    echo %1 es un numero PAR.
) else (
    echo %1 es un numero IMPAR.
)
pause
// %% — en un .bat hay que escapar el módulo como %%. En CMD directamente sería %.

Copia de Seguridad

%1 · %2 · if exist · copy
PROPUESTA
Crea copia.bat con dos parámetros (origen y destino). Comprueba que existen ambos parámetros y que el archivo origen existe. Si todo OK → copia con copy.
Uso: copia.bat origen.txt destino.txt — informa de cada paso al usuario
copia.bat — SOLUCIÓN 
@echo off
if "%1"=="" (
    echo ERROR: Falta archivo origen.
    exit /b
)
if "%2"=="" (
    echo ERROR: Falta archivo destino.
    exit /b
)
if not exist "%1" (
    echo ERROR: El archivo origen "%1" no existe.
    exit /b
)

copy "%1" "%2"
echo Copia realizada de "%1" a "%2".
pause

Menú Interactivo

goto · cls · date /t · time /t
PROPUESTA
Crea menu.bat con un menú de 3 opciones: 1) Mostrar fecha, 2) Mostrar hora, 3) Salir. Si opción inválida → volver al menú.
Usa :menu como etiqueta + goto menu para volver. cls limpia la pantalla antes de mostrar el menú.
menu.bat — SOLUCIÓN 
@echo off
:menu
cls
echo ====================
echo        MENU
echo ====================
echo 1. Mostrar fecha
echo 2. Mostrar hora
echo 3. Salir
echo ====================
set /p opcion="Elige una opcion (1-3): "

if "%opcion%"=="1" (
    echo Fecha actual:
    date /t
    pause
    goto menu
)
if "%opcion%"=="2" (
    echo Hora actual:
    time /t
    pause
    goto menu
)
if "%opcion%"=="3" (
    echo Saliendo del programa...
    exit /b
)
echo Opcion no valida. Intentalo de nuevo.
pause
goto menu
P4

Borrado Múltiple con Confirmación

shift · goto · if exist · set /p
PROPUESTA
Crea borrarmulti.bat que reciba varios archivos como parámetros y los borre uno a uno, preguntando antes de cada borrado: "¿Borrar archivo X? (S/N)"
Clave: shift desplaza los parámetros (%2→%1, %3→%2…). Cuando %1 queda vacío, se han procesado todos.
borrarmulti.bat — SOLUCIÓN 
@echo off
if "%1"=="" (
    echo ERROR: No se introdujeron archivos para borrar.
    exit /b
)

:borrar_siguiente
if "%1"=="" goto fin
if exist "%1" (
    set /p resp="¿Borrar '%1'? (S/N): "
    if /i "!resp!"=="S" (
        del "%1"
        echo "%1" borrado.
    ) else (
        echo "%1" no borrado.
    )
) else (
    echo "%1" no existe.
)
shift
goto borrar_siguiente

:fin
pause
// shift — mueve los parámetros: %2 pasa a ser %1, %3 pasa a ser %2, etc. Cuando no quedan más, %1 está vacío y el bucle termina.
P5

Calculadora Básica (4 operaciones)

%1 %2 %3 · set /a · operadores
PROPUESTA
Crea calculadora.bat con tres parámetros: calculadora.bat 5 + 3 → "Resultado: 8". Soporta +, -, *, /. Error si operador inválido.
Usa cuatro if independientes, uno por operador. Si ninguno matchea → operador inválido.
calculadora.bat — SOLUCIÓN 
@echo off
if "%1"=="" (
    echo Uso: calculadora.bat numero1 operador numero2
    echo Operadores: +  -  *  /
    exit /b
)
if "%2"=="" (
    echo ERROR: Falta el operador.
    exit /b
)
if "%3"=="" (
    echo ERROR: Faltan parametros.
    exit /b
)

if "%2"=="+" set /a res=%1+%3
if "%2"=="-" set /a res=%1-%3
if "%2"=="*" set /a res=%1*%3
if "%2"=="/" set /a res=%1/%3

if "%res%"=="" (
    echo Operador no valido. Usa +, -, * o /
) else (
    echo Resultado: %res%
)
pause
