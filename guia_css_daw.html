01

Cómo incluir CSS en HTML

Hay tres formas de añadir estilos. La más recomendada para proyectos reales es el archivo externo.

Método Sintaxis Dónde va Recomendado
Archivo externo <link rel="stylesheet" href="estilos.css"> En <head> ✓ Proyectos
Bloque interno <style> ... </style> En <head> ~ Prototipos
Inline <div style="color:red"> Atributo HTML ✗ Evitar
html — archivo externo (recomendado) 
<head>
  <!-- Enlaza el archivo externo -->
  <link rel="stylesheet" href="estilos.css">
</head>
02

Selectores CSS

Los selectores determinan a qué elementos del HTML se aplican los estilos.

Estructura HTML y qué selecciona cada selector demo visual
<h1>Título</h1> ← seleccionado por h1
<p id="principal">Texto</p> ← por #principal
<p class="destacado">Destacado</p> ← por .destacado
<div class="caja">Caja</div> ← por div.caja (intersección)
Selector CSS Qué selecciona
Etiqueta p { } Todos los <p> del documento
ID #nombre { } El elemento con id="nombre" (único)
Clase .clase { } Todos los que tienen class="clase"
Universal * { } Todos los elementos del documento
Unión h1, h2 { } Varios selectores a la vez
Intersección div.caja { } Solo <div> que tenga la clase caja
Orden de especificidad: ID > Clase > Etiqueta. Si dos reglas contradictorias aplican al mismo elemento, gana la más específica.

Alineación y Flexbox

Flexbox es la herramienta más poderosa para alinear y distribuir elementos. Practica los valores con el simulador.

css — text-align y flexbox 
/* Alineación de texto */
p { text-align: left | center | right | justify; }

/* Flexbox — activa el modo flex en el contenedor */
.contenedor {
  display: flex;
  justify-content: center;  /* eje horizontal */
  align-items: center;     /* eje vertical */
  gap: 16px;               /* espacio entre hijos */
}

Tamaño y Box Model

Todo en CSS es una caja. El modelo de caja define el espacio que ocupa cada elemento: contenido, padding, borde y margen.

Modelo de caja — visualización interactiva
margin
border
padding
contenido (width × height)
css — box model completo 
.elemento {
  width: 200px;            /* ancho del contenido */
  height: 100px;           /* alto del contenido */
  padding: 10px;           /* espacio interior (entre contenido y borde) */
  border: 2px solid black;  /* borde visible */
  margin: 20px;            /* espacio exterior (separa del resto) */
  border-radius: 10px;    /* esquinas redondeadas */
}

Propiedad display

La propiedad display controla cómo se coloca un elemento en el flujo del documento.

Comportamiento de cada valor de display demo visual
display: block
Ocupa todo el ancho
display: inline
Fluye en línea:  elementoinline
display: inline-block
Respeta w/h:  inline-block
display: flex
item
item
item
Valor Efecto principal
block Ocupa todo el ancho disponible, fuerza salto de línea antes y después
inline Solo ocupa el espacio de su contenido, no acepta width/height
inline-block Fluye en línea pero acepta width y height como un bloque
flex Activa Flexbox en el contenedor — hijos se distribuyen en fila
none El elemento desaparece del documento (no ocupa espacio)
07

Pseudo-clase :hover

La pseudo-clase :hover aplica estilos cuando el ratón pasa por encima del elemento. Usa transition para suavizar el cambio.

Pasa el ratón — cada caja tiene un :hover diferente interactivo
Cambia color
Escala ×1.12
Sube 6px
Brilla en cyan
css — hover con transition 
button {
  background-color: #333;
  transition: all 0.3s ease;  /* suaviza el cambio */
}

button:hover {
  background-color: #e8006f;
  transform: scale(1.1);
}

Propiedades de Fuente

El texto tiene mucha personalidad. Con cuatro propiedades puedes controlarlo completamente.

Combinaciones de fuente en acción
font-size Texto grande 24px font-size: 24px
font-weight Light 300 · Bold 700 · Black 900
font-style Texto en itálica — añade énfasis
font-family Monospace para código y datos
css — propiedades de texto 
p {
  font-family: Arial, sans-serif;  /* fallback si no carga */
  font-size: 16px;
  font-weight: bold;            /* normal | bold | 100–900 */
  font-style: italic;
  line-height: 1.6;             /* interlineado */
  letter-spacing: 0.05em;       /* espaciado entre letras */
}
09

Animaciones CSS

Las animaciones CSS usan dos partes: @keyframes (define los estados) y animation (las activa en el elemento).

Animaciones en vivo — haz clic para activar interactivo
← Selecciona una animación
css — keyframes y animation 
/* 1. Definir los estados */
@keyframes miAnimacion {
  0%   { transform: translateX(0);     background: green; }
  100% { transform: translateX(200px);  background: red; }
}

/* 2. Aplicar al elemento */
.elemento {
  animation: miAnimacion 2s ease-in-out infinite alternate;
  /* nombre · duración · curva · repeticiones · dirección */
}

01

Selector de clase

Fondo amarillo solo a párrafos .destacado
Crea un CSS que ponga fondo amarillo solo a los párrafos que tengan la clase destacado. Los demás párrafos no deben verse afectados.
Usa el selector de intersección con punto: selector de etiqueta + clase.
css — solución 
p.destacado {
  background-color: yellow;
}
02

Botón con hover

:hover · transform: scale() · transition
Haz que un botón cambie a color azul y aumente su tamaño un 10% al pasar el ratón. Añade transición suave.
Usa :hover y transform: scale(1.1). Añade transition: 0.3s al estado normal.
css — solución 
button {
  transition: all 0.3s;  /* va en el estado base */
}
button:hover {
  background-color: blue;
  transform: scale(1.1);
}
03

Caja redondeada

width · border · padding · border-radius
Crea un <div> de 200×200px, con borde negro de 2px, padding de 20px y esquinas redondeadas de 15px.
Recuerda que border-radius redondea las esquinas.
css — solución 
.caja {
  width: 200px;
  height: 200px;
  border: 2px solid black;
  padding: 20px;
  border-radius: 15px;
}
04

Texto centrado con Flexbox

display: flex · justify-content · align-items
Centra un texto dentro de un <div> de 300×150px usando Flexbox. Tanto horizontal como verticalmente.
Necesitas display: flex + justify-content (horizontal) + align-items (vertical).
css — solución 
.contenedor {
  width: 300px;
  height: 150px;
  display: flex;
  justify-content: center;  /* horizontal */
  align-items: center;      /* vertical */
  background: lightgray;
}
05

Animación fade in

@keyframes · opacity · animation
Crea una animación que haga que un cuadrado rojo pase de opacidad 0 a 1 en 2 segundos. La animación debe repetirse en bucle (alternando).
Usa opacity dentro de @keyframes. Para que vaya y vuelva: animation: ... infinite alternate.
css — solución 
@keyframes aparecer {
  from { opacity: 0; }
  to   { opacity: 1; }
}
.cuadrado {
  width: 100px;
  height: 100px;
  background: red;
  animation: aparecer 2s ease-in-out infinite alternate;
}
06

Cuadrado que se mueve y cambia de color

translateX · background en keyframes · infinite alternate
Haz un cuadrado que se mueva 200px a la derecha y cambie de verde a azul en bucle continuo (ida y vuelta).
infinite alternate hace que la animación vaya y vuelva sin cortes abruptos.
css — solución 
@keyframes mover {
  0%   { transform: translateX(0);     background: green; }
  100% { transform: translateX(200px); background: blue; }
}
.cuadrado {
  width: 80px;
  height: 80px;
  animation: mover 2s ease-in-out infinite alternate;
}
